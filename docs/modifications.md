## Modifying the UI 

If you want to modify the UI of the chatbot widget, you edit the respective classes in the [`style.css`](../static/css/style.css)


| Element      | Class        |
|--------------|--------------|
| Chat Window  | .widget      |
| Chat Header  | .chat_header |
| Chats        | .chats       |
| Bot Message  | .botMsg      |
| User Message | .userMsg     |
| Keypad       | .keypad      |
